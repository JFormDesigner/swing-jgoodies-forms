:: Visual Builders ::

Find here a list of visual builder tools that support the JGoodies FormLayout.

The Swing Empire FormBuilder has been designed to work with the FormLayout. It focuses on a separation of the different roles in the UI production process.

JETA Software's Form Builder is integrated with their Abeille cross-platform modeler, query tool and database browser.

XCentric's JAXFront generates dynamic user interfaces based on an XML schema.

The JFormDesigner project plans to support FormLayout, TableLayout and HIGLayout. The software is announced for Q2 2004.

Rat und Tat plans to support the FormLayout in the JBeaver visual designer.
